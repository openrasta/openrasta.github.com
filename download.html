Downloading OpenRasta

OpenRasta has now transitioned to OpenWrap's server. To install OpenRasta, simply make sure you have the nuget repository installed, and from your project, do:

o add-wrap openrasta-full

If you want the 2.0 binaries, they are not currently provided, and you'll have to build that version from source using the instructions in the documentation.
